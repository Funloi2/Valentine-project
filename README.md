This project is a playful "Valentine's" game. It features:
* **Interactive UI:** A persistent "Yes" button and a mischievous "No" button.
* **Dynamic Animations:** The "No" button jumps to random positions when approached by the mouse, while the "Yes" button grows larger with every attempt to click "No."
